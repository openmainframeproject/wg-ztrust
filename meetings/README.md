# WG Meeting notes

All WG meeting notes are stored here and named in YYYY-MM-DD.md format.
 Recordings and other assets from conference calls systems can also
 be stored in YYYY-MM-DD.XXX format.

Meeting notes can be recorded as plain text `.txt`
 or as markdown `.md`. Markdown taggin should be kept simple
 so that meeting minutes can be easily (painlessly) read
 when not rendered.
